# **Deskripsi Project**

Project ini adalah tugas Ujian Tengah Semester mata kuliah **Pemrograman Web 1** yang bertujuan untuk mengonversi desain Figma menjadi website fungsional menggunakan HTML, CSS, dan JavaScript.

Website terdiri dari **3 halaman utama**:

## 1️⃣ **Halaman Login (index.html)**

* Berisi form login (email dan password/NIM)
* Memiliki validasi input menggunakan JavaScript
* Menampilkan animasi loading sebelum masuk dashboard
* Desain monochrome sesuai referensi gambar
* Redirect otomatis ke dashboard setelah login

## 2️⃣ **Halaman Dashboard (dashboard.html)**

* Menampilkan ringkasan data (Total Produk, Total Penjualan, Total Revenue)
* Data diambil dari array summary pada `script.js`
* Menggunakan layout flexbox
* Terdapat tombol menuju halaman data produk

## 3️⃣ **Halaman Data Produk (products.html)**

* Menampilkan tabel data produk (nama, harga, stock)
* Data diambil dari array of objects pada `script.js`
* Fitur Edit (alert) dan Delete (hapus baris tabel)
* Menggunakan manipulasi DOM (remove())
